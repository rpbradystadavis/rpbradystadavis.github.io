About Me

My name is Ryan Brady and I’m a data analyst consultant where I’ve spent the last few years working with clients to try and build solutions that allow clients to better understand their own internal data. In my young data science career; I’ve taken a liking to working with GIS data and building GIS data pipelines as well as exploring different modelling methods to forecast events. Today I work at Control Risks where I’ve grown my Python and R skills but also have utilized BI tools like Tableau allowing me to improve my dashboarding and excel skills.

While working, my favorite projects have included incorporating dashboarding, programming, and statistics to create unique dashboards for clients. Recently, I had the opportunity to build out the shiny applications for the Citibank Monitorship where I picked up AWS/Linux server management skills, leaflet and shiny skills, and unit testing skills. Since I've worked on this project I've started to explore more methods of incorporating Shiny and statistics and created internal dashboards that allows users to call and use scikit-learn functions in python via R Shiny and change the inputs with different widgets.

Some personal things that have kept me sane through the pandemic include a rediscovery of running and hiking and watching too much soccer, Go Wolves!
