Executive Summary

The main goal of this project is to automate classification of crops growing in specific fields in Yolo County, CA, by using visible and infrared light bandwidths. We constructed shell scripts to form a .jp2 file of each light covering Yolo County and then a script that extracts an image for every crop fieldâ€™s bandwidth. After extracting the light bandwidths from two satellite images we implemented a model that correctly classifies crops growing in fields 90% of the time.
